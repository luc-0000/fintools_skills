# Frontend UI/UX Spec Index

This bundle extracts the reusable UI/UX contract from `frontend/` and stores it under `frontend/docs/specs`.

## Scope

- Included: layout shell, route pages, page-level components, active global styles, assets referenced by the running UI
- Excluded: services, stores, models, API semantics, domain workflows, runtime data behavior, `node_modules`
